## 0. 这是什么

3W 是一个**方法学原创**项目：每个单细胞的转录组里同时写着「我是谁、我经历了什么、我在哪」。3W 用**半监督、可识别**的方式把这三根轴从转录组里学出来——不靠人去定义内容，靠数据自己说话——并最终走向 **"以小博大"**：用穿刺针大小的散细胞，重建整块组织的构筑。

**目标刊物**: 主攻 Nature Communications / Genome Biology；冲刺 Nature Methods（取决于可识别性定理）。**不做湿实验**，全部用公共正交数据验证。

> 完整愿景与思维实录见 `docs/3W_VISION.md`；κ 的数学/零模型/可识别性见 `docs/METHOD_DESIGN_kappa.md`。
