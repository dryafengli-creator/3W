# 方法设计 — 跨域保守共调 κ（Conserved Co-Regulation）

> **创建**: 2026-06-23 · **维护**: Qjun + 助手
> **定位**: 这是 paper3 从“跨器官 atlas/FM”转向**纯方法学原创**后的核心方法设计文档。目标刊物：主攻 Nature Communications / Genome Biology，冲刺 Nature Methods（取决于可识别性定理能否证出）。**无湿实验**。
> **真实性原则 (P1)**: 本文档区分【已知】【新提出】【待证 conjecture】。仿真/实验数字在真正跑出来前一律标 `[待跑]`，绝不臆造。

---

## 0. 一句话

我们占住一个领域级痛点——**“跨域的同一种细胞态”被循环定义**（先用以混合域为目标的嵌入把细胞搅在一起，再把那里的簇叫保守态，不可证伪）。为此定义一个**免整合、不需跨域细胞对应**的新变量 **κ（保守共调广度）**，配**保组成零模型 + FDR**，并尝试证其**可识别性定理**。

---

## 1. 问题形式化

- D 个域（器官/条件/数据集/物种/个体），域 d 有 N_d 个细胞、共享基因空间 G（取**测到基因的交集**，避免 HVG 并集零填充污染共调估计）。
- 目标：找出在 ≥τ 个域里**保守**（同一内部共调结构）的细胞态/基因程序，给 calibrated 显著性，**且不先把各域整合进共同隐空间**。

**循环的精确表述**：标准流程学一个嵌入 f，其训练目标是**最小化域可分性**（batch mixing）；随后“保守态”= f 隐空间里的簇。但 f 是被训练去混合域的，所以隐空间的跨域对齐是**被强加**的，不是被推断的 → 在该空间里做的任何保守性检验都循环。M4（XOC+DCL）的塌缩是此循环的实证。

---

## 2. 新变量：κ（保守共调广度）

**核心**：绝不跨域对齐细胞；测“**域内基因共调结构**在多域里独立重现的程度”。共调在每域内单独估计，故不需跨域对应，循环被切断。

对候选程序方向 v ∈ R^|G|：

1. 每域 d 拟合**域私有** nuisance 模型 M_d，吸收：每基因均值（housekeeping）、文库深度、**细胞组成 + 类型内均值**（扣掉“多类型混合→伪基因相关”的 Simpson 混杂）。M_d 不跨域共享任何参数。
2. 残差 r_d（去混杂后的真实共调）。域内相对共调强度：
   ρ_d(v) = Var(vᵀ r_d) / [ (1/|G|)·tr Cov(r_d) · ||v||² ]
   （对单位 v，分母=平均特征值；ρ>1 表示 v 比随机方向承载更多残差共变。）
3. **保守广度 κ**：
   κ(v) = ( Π_{d=1}^{D} ρ_d(v) )^{1/D}      （几何均值；或 min_d ρ_d(v)）
   几何均值/min ⇒ 要求程序在**所有**域在场才算保守，杜绝单域独大。

候选程序基底的构造**不得用跨域信息**（否则又循环）：用各域残差 PCA 方向的并集作候选池，再用 κ 排序、做检验。

---

## 3. 零模型与检验（新，精准打混杂）

**保组成零模型**：按域生成数据，保留 (每基因边际分布、文库、细胞组成与类型内均值)，但**独立打散域内残差以摧毁跨域共调**。在该零模型下，任何 κ 都来自 nuisance/有限样本。由此得 κ 的 calibrated 零分布，对候选程序集做 **FDR 控制**。

> 现有保守性分析（相关 NMF/PCA 载荷、肉眼 UMAP）**没有**保组成零模型与显著性——这是我们的关键缺口填补。

---

## 4. 可识别性定理（待证 conjecture — Nat Methods 杠杆）

生成模型：
  X_d = Σ_k a_{dk} u_k (共享程序，活性 a_{dk} 可为 0) + Σ_j b_{dj} w_{dj} (域私有) + nuisance(M_d) + 噪声。

**命题**：κ 的总体最大化子恢复“出现在 ≥τ 个域”的共享子空间 span{u_k}，且在**不观测任何跨域细胞对应**下可识别（至子空间内旋转），条件：域数足够 / 程序与 nuisance 不共线 / 活性多样性。

- 证到完整可识别性 → 真·新统计结果 → 冲 Nat Methods。
- 只证到检验相合性（consistency）+ 校准 → 仍是扎实 Nat Comms/Genome Biology。
- **诚实概率**：定理 25–35%；“新估计量+新检验”可达 ~70%。

相关理论可借力（但需做出新结果，非照搬）：linked/group factor analysis 可识别性、coupled tensor 分解唯一性、measurement-error/residual 一致性。

## 5. 与最近邻前人工作的边界（防串烧，请拷打）

| 前人 | 做什么 | 缺什么（我们补的） |
|---|---|---|
| MSFA / group factor analysis | 共享+私有高斯因子 | 跨域**联合拟合**(残留循环)；不适配计数/组成；无保守检验/组成零模型/广度估计量 |
| LIGER / iNMF | 整合用 shared/specific metagenes | 无显著性、无可识别性、对 k 敏感 |
| DIALOGUE / scITD | 组织内跨个体多细胞程序 | 非跨域保守估计量；无保组成零模型 |
| IRM / 不变性学习 | 跨环境预测不变性 | 需监督标签；我们无标签且要的是结构不变性 |
| scVI/Harmony 等整合 | 共同隐空间 | 强加对齐 = 循环本身 |

**真正新的四件**：① 域私有 nuisance + 残差化 ⇒ 免整合、切断循环；② 广度估计量 κ（测量而非拟合保守）；③ 保组成零模型 + FDR；④ 可识别性定理。

## 6. 验证program（无湿实验，器官放开）

1. **仿真/可识别性** ✅ **首轮通过（2026-06-23，线性高斯情形）**，脚本 `99_scripts/M_method_conservation/sim_kappa_calibration.py`：
   - **校准**：无共享程序时，FDR 0.05/0.10/0.20 下被判显著比例 = **0.021 / 0.083 / 0.188**（均 ≤ 名义）→ 零模型 calibrated。
   - **恢复 + 抗混杂**：植入 3 个共享程序 + 组成混杂时，本法 top-κ 方向对真实共享子空间捕获 **0.864**、对组成混杂仅 **0.006**。
   - **复现循环病并修复**：朴素法（域内 PCA 载荷相关、不残差化）对组成混杂捕获 **0.599**（假阳）、对真实共享仅 0.260。
   - **v1→v2 方法学洞见（重要）**：首版零模型“按基因跨细胞打散”毁掉了**所有**结构，把私有程序也判显著（无共享时 100% 假阳）。修正：零模型必须**保留每域共调谱、只摧毁跨域基因对应**（按域独立置换基因）；统计量改为**去掉最强域后的截尾几何均值**（要求 ≥2 域在场）。这正是“保守”零假设的正确形式：*各域各有结构，但跨域不对齐*。
   - 局限（诚实）：此为**线性高斯 + 内置混杂**的内部自洽性检验，**非真实计数数据/真实生物学验证**；可识别性定理仍未证（此结果是其线性情形的经验支持）。
2. **阳性对照（跨几十组织：CellxGene/HCA/Tabula Sapiens）** `[待跑]`：召回已知泛组织保守程序（细胞周期、IFN、通用基质/内皮）；已知组织私有程序不被误判保守。
3. **可复现** `[待跑]`：留出数据/独立队列保守判定可复现；整合法不能（定量展示循环失败）。
4. **正交遗传学** `[待跑]`：保守程序基因跨多疾病 GWAS 富集（全干式）。
5. **CKM showcase** `[待跑]`：用现成 8 器官图谱 + M4 负结果当动机；方法主张通用。

## 7. 证伪线（先说死）

命中以下任一 → 降级或转向，不硬撑：
- 保组成零模型下 κ 仍膨胀（校不准）。
- 召不回植入/已知保守程序，或不优于 MSFA/LIGER。
- 保守判定不能留出复现。
- 可识别性定理假，且检验也不相合。

## 8. 两周理论冲刺计划

- D1–D3：跑 §6.1 仿真（校准 + 恢复 + 对照朴素法假阳）。判据：零模型 FDR≈名义、κ 能选出植入共享程序、朴素法在组成混杂上假阳。
- D4–D7：把 ρ_d/κ 的估计写成可扩展实现（残差化用域内聚类条件均值；候选基底用域内 PCA 并集）；在 2–3 个真实组织（CellxGene 下载）上跑阳性对照。
- D8–D10：尝试可识别性引理的纸面证明（先证简化线性高斯情形）。
- D11–D14：基于证据决定——锁“新估计量+新检验”（稳，子刊），还是追定理（赌 Nat Methods）。
